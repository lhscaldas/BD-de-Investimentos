------------------------------------------------------------------------------------------------------------------------ Requisitos aceitos
### Banco de Dados:
1. O banco de dados deverá ser capaz de:
   - Registrar cotas ou valor inicial de cada ativo.
   - Registrar transações de compra e venda de ativos.
   - Atualizar automaticamente a quantidade de cotas ou o valor de cada ativo com base nas transações registradas.
   - Manter o valor de cada ativo atualizado com base no valor da cota, obtido por meio de uma API confiável ou fonte definida.
   - Calcular a rentabilidade para os seguintes períodos:
     - No dia.
     - No mês.
     - Nos últimos 30 dias.
     - No ano.
     - Nos últimos 12 meses.
     - Nos últimos 3 anos.
     - Nos últimos 5 anos.
   - Calcular projeções baseadas em dados históricos para:
     - Os próximos 30 dias.
     - Os próximos 12 meses.

### Funcionalidades para o Usuário:
2. O sistema deverá permitir que o usuário:
   - Registre cotas ou o valor inicial de cada ativo manualmente.
   - Registre transações de compra e venda de ativos.
   - Edite o valor ou a cota inicial de ativos previamente registrados.
   - Edite transações realizadas anteriormente.

### Interface:
3. A interface deverá exibir:
   - O nome do ativo.
   - A classe do ativo.
   - A subclasse do ativo.
   - O banco em que está registrado.
   - O valor atual do ativo.
   - As rentabilidades calculadas para cada ativo.
   - As projeções de cada ativo.
   - Filtros que permitam selecionar ativos por classe ou por banco.
   - O total do patrimônio, considerando os filtros aplicados.
   - O total das rentabilidades, considerando os filtros aplicados.
   - O total das projeções, considerando os filtros aplicados.
   - Um botão para registrar novas transações relacionadas a um ativo específico.
   - Um botão para registrar novos ativos.

### Exportação de Dados:
4. O sistema deverá permitir que o usuário exporte os dados para um arquivo CSV, incluindo:
   - Nome dos ativos.
   - Classe dos ativos
   - Banco onde estão registrados.
   - Valor atual dos ativos.
   - Rentabilidades calculadas.
   - Projeções calculadas.
   - Transações realizadas.
   - Filtros aplicados no momento da exportação.
